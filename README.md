# Nextjs_TrueFeedBack

Nextjs TrueFeedBack project focused on AI SDK with a stack that includes AI SDK, Feedback Workflows, MongoDB, Next.js.

## Tech Stack

- AI SDK
- Feedback Workflows
- MongoDB
- Next.js
- NextAuth
- OpenAI
- React
- Resend
- Tailwind CSS
- TypeScript

## Project Structure

- `emails/`
- `public/`
- `src/`

## Packages And Scripts

- `my-app` at `Nextjs_TrueFeedBack`
  Available scripts: build, dev, lint, start.

## Getting Started

1. Install dependencies in the package you want to run.
2. Create a local `.env` from any generated `.env.example` file.
3. Run the main development script for that package.

## Maintenance Notes

- Generated folders such as `node_modules/`, `dist/`, and `.next/` should stay untracked.
- Secrets should stay in local `.env` files and never be committed.
- Update this README when the project purpose or setup changes.
